## 🎯 Pedagoške metode

### Primjenjene metode učenja:
- **Project Based Learning** - Studenti rade na realnom projektu od Dana 1
- **Hands-on Workshops** - Praktične Figma vježbe svaki sat
- **Peer Feedback Sessions** - Strukturirane kolaborativne sesije
- **Design Thinking Process** - Empathy → Ideate → Prototype → Test
- **Case Studies** - Analiza stvarnih IT projekata
- **Interactive Activities** - Kontinuirani engagement kroz cijelu sesiju

## 📝 Napomene za facilitatora

### Ključni principi:
- Svi materijali **na hrvatskom jeziku**
- **Tabula rasa pristup** - Pretpostavka nultog predznanja
- **Continuous engagement** - Nema "dead time" u sesijama
- **Minute-by-minute planning** - Detaljan raspored aktivnosti
- **Facilitator support** - Detaljne upute i backup planovi

### Format sinopsisa:
Svaki synopsis sadržava:
- ⏰ Minute-by-minute raspored
- 📖 Konceptualna objašnjenja
- 💻 Praktične Figma vježbe
- 👥 Grupne aktivnosti
- 🔄 Backup aktivnosti
- ❓ Predviđena studentska pitanja
- ✅ Kritični checkpointi
